Hubungkan ke Supabase dulu

Aplikasi ini butuh database Supabase agar data tersimpan permanen. Ikuti langkah ini:

  1. Buat project gratis di supabase.com
  2. Buka SQL Editor, jalankan isi file schema.sql yang disertakan
  3. Buka Project Settings → API, salin Project URL dan anon public key
  4. Buka file index.html, cari bagian SUPABASE_URL dan SUPABASE_ANON_KEY di dalam <script>, lalu isi dengan nilai Anda
  5. Simpan, refresh halaman ini

Detail lengkap ada di file README.md.
